The Inspector

Basics

Open the Inspector in Google Chrome by pressing F12 or Ctrl+Shift+C.
You can also right-click on any element and click "Inspect".
This will probably look something like this:

Inspecting elements

  • Elements panel --> entire HTML structure of the page
  • Left of the "Elements" panel are two symbols. Click the left one of these and hover over any element on the page to see the HTML-part of it. Click on any part of the page to see all the Styles that affect it on the bottom right. Edit something there to test changes life.
  • Shift + Ctrl + C --> opens inspector in Element panel
  • Shift + Ctrl + J --> opens inspector in JavaScript console
  • Shift + Ctrl + I --> Open the last panel
  • DOM-tree = Document Object Model tree
  • Right-click in element in the element panel --> Scroll into view --> you jump to that element on the page
  • Ctrl + Shift + p --> command line --> e.g. show ruler ... --> shows ruler on hover. It's recommended to uncheck autocomplete in the settings.
  • Select element --> inspect --> press "h" to hide

View and change CSS

  • Just edit the CSS in the "Styles" panel to edit CSS live - this won't change the source code
  • In the CSS-panel --> ".cls" --> click to add and toggle classes on and of for an element
  • In the CSS-panel --> ":hov" --> toggle pseudostates (for example "hover" --> the element acts like being hovered over
  • press symbol right of the brush -->show Box model --> double click on values to change them

The CSS-overview panel

  • Get an overview of the CSS on the page: Open DevTools (F12) --> press three dots on the top right --> More Tools --> CSS overview
  • Summary --> e.g. the number of elements, types of selectors, ...
  • Colors: all the used colors - also highlights low-contrast texts issues --> click on color --> shows elements that use that color --> click on element --> takes you to the HTML part of the element
  • Font Info --> same as Colors but for fonts
  • Unused Declarations --> does what it says
  • Media Queries --> Lists all media queries on the page
