Basic HTML

This page contains relevant information I regularly looked up, when learning to use HTML

Important Basic-HTML Elements

The most important HTML-elements:

Page-Formatting

  • h1-h6 - Headlines

    Printed bold, gets smaller the higher the number

  • p - paragraph

    After every paragraph comes a new line

  • ol and ul - ordered list and unordered list

    numbered or simply dots

  • li - a list element

    for every item in the list - comes nested into the ul/ol elements

  • div - for grouping and adding classes to blocks of elements

    block level by default; commonly used as a container element --> to divide the page into different blocks and apply styling to these blocks

  • span - for grouping text together

    group text content and inline HTML elements for styling - should only be used when no other semantic HTML element is appropriate

Text-Formatting

  • strong - important text, printed bold
  • em - emphasized text - printed italic

    an emphasized word should have a different meaning as normally

  • i - italic text

    different mood or "voice" than the non italic word

  • small - smaller text
  • del - deleted text
  • ins - inserted text
  • sub - subscript text
  • sup - superscript text
  • mark - marked text

Other HTML-Elements

  • div - just an empty container
